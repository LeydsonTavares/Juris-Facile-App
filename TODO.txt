Fazer sempre que realizar alterações no app.

git add . //adiciona todas as mudanças realizadas no app

git commit -m “commitando” //commita as mudanças

git push –set-upstream origin master //sobe as mudanças no github

git push ionic master //sobe as mudanças no Ionic Pro

git add .


Tudo o que você precisa fazer para enviar uma nova versão para Testadores é:

git add -p 
git commit -m "Minhas novas mudanças" 
git push iônico * mestre * // mestre é qualquer ramo que você gostaria de empurrar
